About Our Final Project

Topic: Airbnb Price Predictor Using SF Homes Data

Reason: This could be useful for anyone who has property and wants to monetize it. Once this code is finished, it could be applied to predicting prices in other cities as well.

Description: Dataset was downloaded from http://tomslee.net/category/airbnb-data. The dataset is a csv that contains several features that may determine Airbnb rental prices.

Question: What should we price our Airbnb home?

Goal: To predict the price range with machine learning algorithm based on the dataset provided and store the dataset into PostgreSQL. If time is permitted, we will deploy the machine learning model into a webapp either using Flask or streamlit.

Plan:

  • We will try to use linear regression to predict price range (module 17)
  • Machine learning to train the models (module 18-19)
  • Store clean data to PostgreSQL database
  • Manipulate the csv file

Tools:

  • AWS (maybe)
  • Postgresql
  • flask/json/streamlit(if time is permitted)
